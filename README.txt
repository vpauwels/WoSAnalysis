Run the scripts in the following order:
- paper_lists.py : Downloads the lists with the papers for each journal and year.
- download_IF_data.py : Downloads the html files with the raw data of the Impact Factor calculation.
- extract_IF_data.py : Extracts the raw data from the html files.
- download_citations.py : Extracts the citation record for each paper.

The .csv files are the input files for these scripts.
